# API keys needed
- TfL app_key for Unified API.
- Google Maps API key only if using Google Maps/Places/Routes; OSM basic tiles do not need a key but must follow OSM policy.
- Mapbox/ORS optional for geocoding/routing at larger scale.
- TravelSheep: booking/cottage provider affiliate/API, RapidAPI/Skyscanner optional, weather API.
- SeaLife: property data provider access; portal scraping should be avoided unless terms allow it.
- Linode token/object storage for backups.
- LLM keys: OpenAI, Anthropic, Gemini, Grok; Ollama local URL.
