# Geck0verse Core Platform

## Purpose

Geck0verse is Phil's personal operating platform.

It connects:
- Crumble
- CyberX
- Geck0 Search
- Knowledge Graph
- Wiki
- Git
- APIs
- Dashboards
- BlackGeck0
- GreenGeck0
- PinkGeck0
- BlueGeck0
- YellowGeck0
- PurpleGeck0
- Personal Geck0 services

## Core Principle

Everything should be modular.

Every tool, app, dashboard, bot, scanner, voice service, reminder system, file vault, note app, photo processor and AI workflow should plug into the same shared platform.

## Standard Menu Controls

All terminal menus should support:

q = back / quit current screen
x = exit completely
h = help
s = search
r = refresh
m = main menu
l = logs

## Naming Rule

All visible menus and titles must use Geck0 with a zero.

Correct:
- Geck0verse
- BlackGeck0
- GreenGeck0
- Geck0 Search
- Geck0 Remind

Avoid:
- Gecko
- GeckoVerse
- Black Gecko

## First Build Target

The first working version should run locally on JelliedEel.

ThaiGreen should mainly host GeckoScan / BlackGeck0 scanning tools, with results pushed back into JelliedEel.
