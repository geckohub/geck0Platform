# Geck0 Keyboard + Command Palette

## Purpose

Geck0 Keyboard is the fast launcher for the full Geck0verse menu.

It should work like a command palette.

Main idea:

- Press a keyboard shortcut
- Full Geck0 menu opens
- Type commands naturally
- Crumble / CyberX / Geck0 Search route the request

## Default Shortcut

Preferred shortcut:

Ctrl + Space

Alternative shortcuts:

Alt + Space
Ctrl + Shift + G
Super + G

## Behaviour

When shortcut is pressed:

1. Open Geck0 command palette
2. Focus search/input bar
3. Allow natural commands
4. Show matching actions
5. Execute selected action
6. Allow toggle into full dashboard

## Menu Integration

The command palette should be connected to:

- Crumble
- CyberX
- Geck0 Search
- Main Geck0 menu
- Full dashboard
- Wiki
- Git
- API Gateway
- Knowledge Graph

## Dashboard Integration

Clean view:

- Search bar doubles as command palette
- Mic button sends voice command to Crumble
- Upload button sends file to Geck0 Files / Photos / Voice / Search

Full dashboard view:

- Shortcut opens menu overlay
- Search can find actions, files, notes, modules, devices, logs and dashboards

## Terminal Controls

q = back / quit current screen
x = exit completely
h = help
s = search
r = refresh
m = main menu
l = logs
