# Geck0 Search Architecture

Version: 0.1
Platform: Geck0verse

------------------------------------------------------------
Purpose
------------------------------------------------------------

Geck0 Search is the universal search engine for the entire
Geck0verse ecosystem.

The goal is that the user never needs to remember where
information is stored.

Instead they simply ask:

    search ketama

or

    search meeting yesterday

or

    search mmcy.uk

Geck0 Search discovers the information automatically.

------------------------------------------------------------
Core Principles
------------------------------------------------------------

• Search everything.

• Every module becomes searchable.

• Modules register themselves.

• CyberX ranks results.

• Crumble presents them naturally.

• Knowledge Graph adds relationships and context.
