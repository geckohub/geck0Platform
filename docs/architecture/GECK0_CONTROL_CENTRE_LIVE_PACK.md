# Geck0 Control Centre Live Pack

Adds modular Python files for:

- NLP-ish command router
- live scan process visibility
- latest scan results command
- mini firehose from JE/TG
- clickable menu groundwork
- themes
- gecko banners
- system telemetry

Useful commands:

```text
scan trex mmcy.uk
show latest scan results
firehose
running scans
logs
search ketama
theme matrix
git status
```
