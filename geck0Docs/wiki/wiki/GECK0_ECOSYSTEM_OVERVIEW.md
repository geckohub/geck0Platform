# Geck0 Ecosystem Overview

## Purpose

Geck0 is a modular ecosystem of AI, mapping, cybersecurity, automation, media, hardware, dashboards, and knowledge tools.

The long-term goal is to create an interconnected platform where every project can share data through a common knowledge layer and be queried through Crumble.

## Core branches

### Black Geck0
Cybersecurity, scanning, exploitation validation, reporting, intelligence.

Key projects:
- GeckoScan
- GeckoExploit
- GeckoReport
- Geck0Stealth
- Black Geck0 Security Platform

### Pink Geck0
Automation, AI experiments, creative tooling, digital product building.

Key projects:
- BreadCrumbs
- Geck0 Earth / Geck0 Map
- Alternative Timelines
- TravelSheep
- SeaLife
- Pulse

### Green Geck0
Testing, QA, validation, safe deployment checks.

### Blue Geck0
Media, presentations, videos, creative production.

### Yellow Geck0
Dropshipping, storefronts, RobotRainforest.

### Purple Geck0
Public-facing communication, community, channels, Discord/Telegram/WhatsApp bots, podcasts, YouTube output.

## Core infrastructure

### JelliedEel
Main home server and hub. Hosts dashboards, automation, docs, Git, Crumble, BreadCrumbs, Geck0 Earth, and knowledge systems.

### ThaiGreen
Kali/security node. Runs GeckoScan, GeckoAir support, security experiments, and authorised testing workflows.

### Crumble
Conversational and voice interface across the ecosystem.

### Geck0 Knowledge Graph
Shared observation/intelligence store for all Geck0 systems.

## Unifying principle

Every tool should produce structured observations, not just loose text files.

Pipeline:

Collectors -> Parsers -> Normalised Observations -> Knowledge Graph -> APIs -> Dashboards -> Crumble -> Reports / Videos / Actions
