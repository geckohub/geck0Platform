# Codex Instructions for Geck0

You are working on the Geck0 ecosystem.

## Rules

- Prefer small, modular changes.
- Do not delete existing working files without moving them to archive.
- Add tests for new functionality.
- Update wiki docs for major changes.
- Use manifests for plugins and update ZIPs.
- Keep JelliedEel and ThaiGreen responsibilities separate.
- Treat GeckoScan as authorised-security tooling only.
- Keep Geck0 Earth layers plugin-based and toggleable.
- Use structured observations for new data sources.
- Make changes in small commits.

## Project priorities

1. Stabilise repo structure.
2. Add shared Knowledge Graph foundation.
3. Build Geck0 Earth v1 foundation.
4. Integrate Crumble command router.
5. Modularise GeckoScan.
6. Add BreadCrumbs video/story export pipeline.
