# Suggested official data sources

- Amadeus Self-Service APIs: https://developers.amadeus.com/self-service/apis-docs
- Open-Meteo: https://open-meteo.com/en/docs
- Ticketmaster Discovery API: https://developer.ticketmaster.com/products-and-docs/apis/discovery-api/v2/
- USGS earthquake feeds: https://earthquake.usgs.gov/earthquakes/feed/v1.0/geojson.php
- OpenStreetMap tile policy: https://operations.osmfoundation.org/policies/tiles/

Review current terms, quotas and attribution before production use.
