# Crumble Android

Native Kotlin/Jetpack Compose source for Pixel 8 and Android 8+ devices. Includes text chat, Android speech recognition, text-to-speech, domain list, configurable JE URL/API token, upload API contract and dashboard hand-off foundation.

Open in current Android Studio, install SDK 37 and JDK 17, sync, then run. Use Tailscale or a trusted LAN instead of exposing the Hub directly to the internet.
